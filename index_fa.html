مانیتور دما و رطوبت با ESP32

این پروژه یک کاربرد اینترنت اشیاء (IoT) است که از میکروکنترلر ESP32 برای مانیتورینگ دما و رطوبت استفاده می‌کند. داده‌ها با استفاده از سنسور DHT21 جمع‌آوری شده و در صفحه وبی که روی ESP32 میزبانی می‌شود، نمایش داده می‌شود. علاوه بر این، رابط وب به کاربران امکان کنترل لامپ و دانلود داده‌های ضبط شده را می‌دهد. 📊📉

ویژگی‌ها 🌟

  • اتصال WiFi: اتصال به شبکه WiFi مشخص شده. 📶
  • سنسور DHT21: خواندن داده‌های دما و رطوبت. 🌡️
  • وب سرور: میزبانی یک صفحه وب برای نمایش داده‌های سنسور و کنترل لامپ. 💻
  • SPIFFS: ذخیره داده‌ها به صورت محلی روی سیستم فایل ESP32. 🗄️
  • JSON: استفاده از JSON برای تبادل داده بین ESP32 و رابط وب. 🔄

سخت‌افزار مورد نیاز 🛠️

  • برد ESP32
  • سنسور DHT21
  • رگولاتور ولتاژ 3.3V
  • مقاومت‌ها و خازن‌ها
  • برد بورد و سیم‌ها

نرم‌افزار مورد نیاز 💻

  • Arduino IDE
  • کتابخانه‌های ESP32 و DHT

راه‌اندازی 🚀

  1. اتصال سنسور DHT21 به ESP32.
  2. نصب کتابخانه‌های مورد نیاز در Arduino IDE.
  3. آپلود کد به ESP32.
  4. باز کردن صفحه وب برای مشاهده داده‌ها و کنترل لامپ.

توضیح کد 📜

کد این پروژه به زبان C++ نوشته شده و به قسمت‌های زیر تقسیم می‌شود:

  1. تعریف متغیرها: تعریف متغیرها و کتابخانه‌های مورد نیاز.
  2. راه‌اندازی: تنظیمات اولیه شامل اتصال WiFi و راه‌اندازی سنسور.
  3. حلقه اصلی: خواندن داده‌های سنسور و به‌روزرسانی صفحه وب.

نتیجه‌گیری 🏁

این پروژه یک راه‌حل ساده و کارآمد برای مانیتورینگ دما و رطوبت با استفاده از ESP32 ارائه می‌دهد. این پروژه نه تنها داده‌های محیطی را نمایش می‌دهد بلکه امکان کنترل دستگاه‌های متصل را نیز فراهم می‌کند. ✅
